# Сетевые «Крестики-нолики» (WebSocket, Python)

Веб-приложение для игры «Крестики‑нолики» в реальном времени:

- комнаты (создание/подключение),
- авто‑распределение ролей X/O,
- сервер контролирует очередность и валидирует ходы,
- проверка победы/ничьей после каждого хода,
- обмен данными по WebSocket.

## Запуск

```bash
uvicorn main:app --reload
```

Открыть в браузере: `http://localhost:8000`

## Протокол WebSocket

Подключение: `ws(s)://<host>/ws`

Первое сообщение клиента (обязательно):

- создать комнату:
  - `{ "type": "join", "action": "create", "player_id": "optional" }`
- подключиться:
  - `{ "type": "join", "action": "join", "room_id": "<id>", "player_id": "optional" }`

Ход:

- `{ "type": "move", "cell": 0..8 }`

Серверные события:

- `joined`: назначение роли и ID комнаты
- `state`: состояние доски/очередности/результата
- `error`: ошибка (не ваш ход, занятая клетка, неверные данные и т.п.)
